# Code Collector

一个简单的 Python 工具，用于递归遍历指定目录下的 `.py`、`.cpp`、`.md`、`.txt` 等文件，
读取它们的内容，并将所有内容统一写入一个 `.txt` 文件中，适用于归档、阅读或训练数据整理等场景。

---

## 🧩 项目功能

- 支持递归遍历文件夹；
- 支持多种文件扩展名；
- 支持自动整理输出格式；
- 支持在命令行中指定目录与文件类型。

---

## 📦 使用方法

1. 安装 Python 3（推荐 3.6+）；
2. 将 `main.py` 放入任意目录；
3. 运行：

```bash
python main.py
