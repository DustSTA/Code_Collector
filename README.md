# Code Collector (代码收集器)

一个基于 PyQt6 开发的图形界面工具，主要用于递归遍历指定的目录，并将该目录下各种代码和文本文件的内容提取并合并到一个单一的 `.txt` 文件中（默认输出为 `All_Code.txt`）。这对于项目归档、代码审查、或者为大语言模型（LLM）提供项目上下文非常有用。

---

## 🧩 核心功能

- **🎨 现代图形用户界面**：简洁明了的 PyQt6 界面操作，告别频繁在命令行输入复杂参数。
- **📁 支持文件拖放**：可以直接将目标文件夹拖入输入框，快速定位路径。
- **🔍 智能类型扫描**：选择目录后，应用会自动扫描并找出该目录中实际存在的常见代码文件类型，按需展示相关勾选项。
- **✅ 丰富的多语言支持**：原生支持归档 Python、C/C++、Java、JS/TS、HTML/CSS、Go、Rust、C#、PHP、Vue、Markdown 等十余种主干语言与配置文件。
- **🏷️ 自定义扩展名**：除了常见的语言，还支持手动输入任何你想提取的特定文件后缀（例如：`.ini`, `.env`）。
- **📂 自定义输出目录**：你可以将汇总生成的大文本输出到默认目录，也可以灵活指定保存位置。
- **📝 优雅的内容排版**：写入时会自动为每个文件添加相对路径标题（例：`《src/app.py》`）以及行分隔符，内容排板清晰易读，无惧任何乱码。

---

## 📦 环境依赖

项目基于 Python 运行，需要安装 `PyQt6` 依赖包：

```bash
pip install PyQt6
```

---

## 🚀 使用方法

1. 安装 Python 3（推荐 3.8及以上版本）；
2. 安装必须的 `PyQt6` 库；
3. 运行代码：

```bash
python main.py
```

4. **图形界面操作步骤**：
   - **要遍历的目录**：点击“选择目录”或直接将你想搜集的本地文件夹拖拽至输入区。
   - **输出文件夹**：选择生成的提取文件存放目录（未填写默认与目标目录相同）。
   - **勾选提取的代码类型**：等待目录扫描完毕后，勾选你想提取的内容；若没有对应复选框，可以在下方的“其他文件后缀名”中补充输入。
   - **开始提取代码**：点击执行按钮，等待系统自动提取完毕，结果将在输出目录下生成以 `All_Code.txt` 命名的长文本文件。

## 🎯 常见应用场景

- 需要将当前庞杂的项目代码发送给 ChatGPT/Claude 等 AI 辅助查阅，由于文件太多不得不手动挨个复制？使用此工具可以一键打包所有目标源码给 AI！
- 对个人复杂大项目进行统一的源码快照归档与整理。
- 提取指定语言片段以建立项目文档或者阅读离线备份。
